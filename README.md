E-Commerce Website Using MERN Stack
--------------------------------------

A fully functional E-Commerce web application built using the MERN stack (MongoDB, Express, React, Node.js).

Features
----------

  1- User Authentication and Authorization (Login, Register).
  
  2- User Roles (Admin and Customer).
  
  3- Product Management (Add, Update, Delete products - Admin only).
  
  4- Shopping Cart functionality.
  
  5- Payment Integration.
  
  6- Order Management.
  
  7- User Profile Management (Update profile and view orders).
  
  8- Responsive design for all devices.
